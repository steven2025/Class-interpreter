# 多语课堂实时字幕系统（静态前端原型）

这是一个可直接部署到 GitHub Pages 的前端原型，包含：

- 教师端：开课、自动进入发射模式、生成二维码、术语表读取
- 学生端：扫码加入、双语字幕显示、语言切换、历史记录、笔记保存
- 内置模拟模式：无需后端即可先跑通演示

## 本地打开

直接用浏览器打开 `index.html` 即可。

更推荐使用本地静态服务：

```bash
python3 -m http.server 8080
```

然后访问：

```text
http://localhost:8080
```

## 接腾讯云前先做什么

1. 先确认教师端和学生端布局是否符合你的课堂需求。
2. 用模拟模式验证扫码、手机访问和字幕显示。
3. 再把 `js/config.js` 中的 `apiBase` 和 `wsUrl` 替换成腾讯云地址。
4. 把 `teacher.js` 里“手动发送字幕”的逻辑替换成“把音频发到云端后接收转写结果”。

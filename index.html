多语课堂实时字幕系统

适合部署到 GitHub Pages 的前端原型。教师端负责开课与生成二维码，学生端扫码后在手机或平板浏览器中接收双语字幕。

进入教师端 打开学生端

当前版本能做什么

  • 教师创建课堂并生成学生二维码
  • 学生扫码或打开链接直接进入课堂
  • 支持上下双语字幕显示
  • 支持语言切换、历史记录、存为笔记
  • 内置“模拟实时模式”，便于你先做前端演示

后续接腾讯云时替换什么

  • 将模拟推送替换为 WebSocket / API 网关
  • 将手动发送字幕替换为 ASR 实时转写
  • 将本地术语表替换为云端术语表
  • 将本地笔记替换为云端保存
